Lecture 1: Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It is used to structure content on the web.

Basic Structure

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>+
  <body>
    <h1>Hello World</h1>
  </body>
</html>

Key Points

  • HTML uses tags like <h1>, <p>, <a>.
  • Tags usually come in pairs: opening and closing.
  • Always start with <!DOCTYPE html>.
